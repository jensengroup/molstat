Setup Guide

Before we start with week 1, we need to make sure that everything is installed correctly. Follow the guides belove for your chosen operating system, and check if everything is working by writing the following in a terminal.

First open python

python

Then type the following in the active shell

>>> import numpy
>>> import pylab

If this is successful with no error messages, then you have successfully setup Python with Numpy and Matplotlib.

Setup for the most well-known operating systems

Virtual Box

Down and install the following programs

  • VirtualBox
  • VirtualBox Extension

Next is to download the virtual harddrive containing Ubuntu 12.04 and open it. This will setup the virtual box.

Linux / Ubuntu

Install the following packages using apt-get

python-numpy
python-scipy
python-matplotlib

Microsoft Windows

The easiest way to get all the packages needed are to install the Continuum's free Anaconda package

Download Anaconda

After this is install, setup the anaconda in your $PATH environmental variables
