Welcome to Molecular Statistic

The course includes a series of exercises and the necessary theory. Computer simulation exercises builds on the understanding of statistical mechanics, as students progress in Nano-thermodynamic and using molecular dynamic simulation methods, which are also presented in the course. Molecular dynamic simulations used to solve motion equations for many interacting particles.

Aims:

  • The student must be able to plan, implement and use python packages
  • The student should be able to use python to simulate nanosystems
  • The student must be able to understand and apply selected mathematical algorithms, such as molecular dynamics methods
  • The student must be able to both orally and in writing explain methods used and results obtained
